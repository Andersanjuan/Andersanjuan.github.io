Cellular Automaton Simulator

How it works

Conway’s Game of Life, a 2D cellular automaton.

  • Click a cell to toggle it (alive/dead).
  • Start: run continuously.
  • Step: advance one generation.
  • Random: fill the grid with random live cells.
  • Clear: reset all cells to dead.
